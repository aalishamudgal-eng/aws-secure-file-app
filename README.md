# AWS Serverless File Upload & Viewer App

## 📌 Overview
A production-style serverless application that allows users to upload and view files using AWS services.

## 🚀 AWS Services Used
- Amazon S3 (file storage)
- AWS Lambda (backend logic)
- API Gateway (API handling)
- IAM (security)

## ⚙️ Features
- Upload files to S3
- Fetch and display uploaded files
- Fully serverless architecture
- Secure and scalable design

## 🧠 Architecture
Frontend → API Gateway → Lambda → S3
